GAPCars
________

Ejercicio para proceso de selección para trabajar en GAP. 2017.

Construido utilizando PHP Laravel y AngularJS.

Puedes ejecutar este sitio web utilizando PHP Artisan en la linea de comandos:

php artisan serve

Componentes importantes:

 +- /public - Directorio web raiz
 +- /resources/views/blank.blade.php - Plantilla Laravel maestra que contiene las referencias a los scripts.
 +- /public/js - Archivos de aplicación angularJS GAPCars.
